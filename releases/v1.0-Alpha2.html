The next major update to MFF is here!

What's new:

  • Major overhaul to the Forcefield Projector & GUI
    • Power required has changed
    • Players cannot dig underneath a forcefield
    • Gui is now tabbed
      • Info tab
        • Displays power stored
        • ... power usage
        • ... ff size
        • ... owner
        • ... uptime
      • Sizing tab
        • Sizing is now done with sliders
        • There is a limit on how big you can make a forcefield (without cheating)
      • Security tab
        • Players can be assigned to the security system
        • You can create groups in the security system
        • Players can be assigned to groups
        • There is no other purpose of this tab (yet)
      • Upgrades tab
        • Doesn't exist yet. Don't try clicking it - nothing will happen.
      • Power statistics tab
        • Doesn't exist yet. Don't try clicking it - nothing will happen.
  • New machines
    • Crystal Refinery
    • Used to convert raw crystals (now found underground) into refined crystals
    • A crystal takes 2 minutes to process by default
    • Crystal Constructor
    • A custom crafting mechanic for MFF
    • It's basically a crafting table that takes time to craft
    • Alurite ingot + redstone = alurite base
    • Emerald + refined ruby + refined amethyst + diamond = crystal sheet
  • Crystals
    • Amethyst and ruby ore are now found underground
    • Mining them with an iron or better pickaxe will drop 1 - 4 raw crystals
    • Those crystals can be processed through a crystal refinery to convert them into refined crystals
  • Upgrades
    • Added speed upgrade
    • +100% Speed
    • +25% Power Usage
    • Added efficiency upgrade
    • -50% Power Usage
  • NEI Plugin
    • If NEI is detected, MFF will load its NEI plugin
    • New NEI subset: Modular Fluxfields
    • NEI will show recipes and usages for crafting recipes done with the crystal refinery and crystal constructor
    • NEI will display the time it takes to craft these items
  • Misc
    • Added alurite ingot
