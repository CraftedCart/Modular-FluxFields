  2. Manual Dependency Installation
You shouldn't need to manually download dependencies as Modular FluxFields should download and load them for you. If Modular FluxFields cannot download them however, you will need to do so manually

Getting a download failed message when trying to launch Minecraft? You've come to the right place!

Modular FluxFields requires slick-util.jar to function. Download that from here and save it somewhere. Then goto your Minecraft mods folder, and create a new folder named CraftedCart. Move the slick-util.jar file into that folder, so that it is in mods/CraftedCart/slick-util.jar.

Launch Minecraft, and you should be set!
