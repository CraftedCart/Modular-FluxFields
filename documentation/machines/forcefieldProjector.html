Forcefield Projector

The Forcefield Projector has the ability to create and sustain a forcefield protecting your base

The main feature of the mod, the ability to create and sustain indestructible protecting forcefields, is contained within this block. This block requires power to function, a lot of it. Place a Power Cube on top of the projector to supply it. The forcefield projector has enough internal power storage to store a whole power cube's worth of energy, so don't be surprised when it drains the whole thing.

GUI

The Forcefield Projector's GUI is split up into various tabs. They are:

  • Info
  • Sizing
  • Security In development
  • Upgrades In development - Currently non-existent
  • Power Statistics
In all of the GUI screens, the current power level is displayed at the top.

Info

Forcefield Projector: Info GUI


Data displayed on the info screen includes:

  • The current amount of power stored
  • The power usage per tick
  • The size of the projected forcefield
  • Who owns the Forcefield Projector
  • How long the forcefield has been up for
  • How long the forcefield will take to initialize

There's an easter egg to be found on this GUI screen. See if you can figure it out or expand the spoiler to find out (Hint: Inspect the hierarchy view with F3 for another hint).

Easter Egg Spoiler

Enter the Konami Code (Up, Up, Down, Down, Left, Right, Left, Right, B, A) and a MIDI file will start playing. Note that it may say that the song is playing whilst Java is loading the song. Exit out of the GUI to stop it.

The song is Open Your Eyes by Aviators (Tyler Shaw). Please don't judge it based on the visuals...

Aviators composes and remixes nice music. If you're interested, you can find his music on YouTube, BandCamp, SoundCloud and in various other places (Search for Sound Of The Aviators).

Sizing

Forcefield Projector: Sizing GUI


Here is where you can set the size of the forcefield to be projected. Drag the sliders to set the X, Y, and Z offset relative to the Forcefield Projector block. Using Minecraft's debug screen (F3) will help a lot in figuring out the sizing.

Security In development 

Forcefield Projector: Security GUI


Here, you can manage the security system of the forcefield. A security upgrade will be needed for this to function, but as there is no upgrade system for the Forcefield Projector yet, this requirement is overridden. Players and permission groups can be added using the two text fields at the top left. By default, players are added to the "Everyone" group (Referenced internally as gui.mff:everyone). You can assign players to other groups by selecting players and one group, followed by clicking the "Assign to the selected group" button. Groups are assigned permissions by selecting groups, followed by selecting permissions on the left under the "Group Options" heading.

Tips

  • Hold down Command (OS X) or Control (Other OSs) to select multiple players or groups.
  • Press Command Z (OS X) or Control Z (Other OSs) to undo your last action
  • You can rename multiple groups at once with the same name. Numbers will just be appended to the end of each name.
  • You may not want to be using the default Minecraft window size when working with the security system. Text usually overlaps or is cut off.

The security system is currently a work in progress...

Power Statistics

Forcefield Projector: Power Statistics GUI


Here, you can see the power usage of the Forcefield Projector over time. It keeps track of power usage every tick for the past minute, as well as every second for the past half hour.
