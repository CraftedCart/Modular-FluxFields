Forcefield Projector

The Forcefield Projector has the ability to create and sustain a forcefield protecting your base

The main feature of the mod, the ability to create and sustain indestructible protecting forcefields, is contained within this block. This block requires power to function, a lot of it. Place a Power Cube on top of the projector to supply it. The forcefield projector has enough internal power storage to store a whole power cube's worth of energy, so don't be surprised when it drains the whole thing.

GUI

The Forcefield Projector's GUI is split up into various tabs. They are:

  • Info
  • Sizing
  • Security In development
  • Upgrades In development - Currently non-existent
  • Power Statistics In development - Currently non-existent
In all of the GUI screens, the current power level is displayed at the top.

Info

Forcefield Projector: Info GUI


Data displayed on the info screen includes:

  • The current amount of power stored
  • The power usage per tick and per second
  • The size of the projected forcefield
  • Who owns the Forcefield Projector
  • How long the forcefield has been up for

Sizing

Forcefield Projector: Sizing GUI


Here is where you can set the size of the forcefield to be projected. Drag the sliders to set the X, Y, and Z offset relative to the Forcefield Projector block. Using Minecraft's debug screen (F3) will help a lot in figuring out the sizing.

Security In development 

Forcefield Projector: Security GUI - Managing players


Here, you can manage the security system of the forcefield. A security upgrade will be needed for this to function, but as there is no upgrade system for the Forcefield Projector yet, this requirement is overridden. There are two tabs (found at the bottom) to this GUI screen, manage players mode (the person icon), and manage groups & permissions mode (the padlock icon).

In manage players mode, players can be added to the system, assigned a group and removed. Adding players is as simple as entering their username into the text field (make sure they're online) and hitting return, selecting them, clicking "Add to group" and then selecting a group to assign them to. Deleting a player is done by selecting their username on the sidebar, followed by pressing "Remove". There is no warning dialog; once you've deleted them, they're gone and automatically assigned to the "Everyone" group.

In manage groups mode, you can create groups by entering its name into the text field and hitting return. You will have the ability to set permissions on a group, however the security system is still a work in progress and as such, this feature doesn't exist yet. Removing a group is done by selecting it, followed by pressing "Remove". The only group that can't be removed is the "Everyone" group. As the name implies, the "Everyone" groups is used to manage permissions for all players not on the security system. As with manage players mode, there is no warning dialog; Make sure you're certain when deleting a group, as you will have to reassign each player in that group to a new group.
