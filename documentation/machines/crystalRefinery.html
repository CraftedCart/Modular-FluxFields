Crystal Refinery

The Crystal Refinery is needed to convert those crystals you find underground into something more useful

The crystal refinery is a machine required to produce refined crystals, which are needed to craft several components within Modular FluxFields. Raw crystals are fed into the machine, and after 120 seconds (2400t) (You can, and will probably want to speed this up with speed upgrades), a refined crystal is outputted. You can automate this by pumping items in and extracting items out from any side.

The crystal refinery requires power to run. Place a Power Cube above it and it will draw power from that.

Processing

The crystal refinery can process the following items.

Input Output
Raw Amethyst Refined Amethyst
Raw Ruby Refined Ruby
