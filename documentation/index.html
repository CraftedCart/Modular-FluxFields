Modular FluxFields

Official Documentation

  1. Documentation

Modular FluxFields is a Minecraft mod that adds forcefields so that you can protect your base from intruders. Don't expect it to be easy however, creating and maintaining a forcefield is costly.

This mod is still in development! A lot more features are planned, and at the moment, it is impossible to create a forcefield in survival (without cheating).
